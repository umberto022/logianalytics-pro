# 🚚 LogiAnalytics Pro

Plataforma de Análisis Logístico Inteligente para optimizar operaciones de transporte y distribución.

## 🚀 Características Principales

- **📊 Dashboard Inteligente**: KPIs en tiempo real y métricas de rendimiento
- **🚛 Optimización de Rutas**: Algoritmos avanzados para reducir costos y tiempo
- **📈 Reportes Automáticos**: Generación automática de reportes ejecutivos
- **🔔 Sistema de Alertas**: Notificaciones inteligentes para optimización
- **👥 Gestión de Usuarios**: Sistema de autenticación y roles
- **📱 Interfaz Responsive**: Diseño moderno y fácil de usar

## 🛠️ Tecnologías Utilizadas

- **Frontend**: Streamlit
- **Backend**: Python
- **Base de Datos**: SQLite
- **Visualización**: Plotly, Folium
- **Análisis**: Pandas, NumPy, Scikit-learn

## 📦 Instalación

1. Clona el repositorio
2. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```
3. Ejecuta la aplicación:
   ```bash
   streamlit run app.py
   ```
